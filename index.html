Metropolia frameword for 2D game development.

The project is run by second year students from the Metropolia AMK in Espoo. The purpose is to learn software development and game development building a mini framework for 2D games. The framework uses C# and .NET framework with OpenGL and TaoClassic.

Purpose and Use

The main goal is to learn how to develop software. Nonetheless, if successful, the framework may be used for further purposes like thesis topic or commercial release.

Stages

At the time of writing (09/11/2013) the framework has just begun and only include the basics of the application and some parts of the math engine.

Documentation
